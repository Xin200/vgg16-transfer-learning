# vgg16-transfer-learning
VGG16 transfer learning project with comparative experiments

This project applies a pre-trained VGG16 model to a 37-class image classification task using transfer learning.

## Experiments
- Baseline model (VGG1)
- Dropout comparison
- Fine-tuning comparison
- Learning rate comparison

## Main findings
- Fine-tuning improved performance
- Higher dropout brought only marginal improvement
- Excessively high learning rate caused optimization failure
